Curso de Excel Básico

Aprende Excel de manera práctica y eficiente con ejercicios reales!

Domina las herramientas esenciales de Excel para el trabajo, estudios o emprendimiento. Desde conceptos básicos hasta funciones avanzadas, con enfoque 100% práctico.

¡Quiero Inscribirme!

Descripción del Curso

Este curso te enseña a manejar Excel básico hasta que logres obtener un nivel óptimo y obtengas la confianza que necesitas con el uso de esta poderosa herramienta, con enfoque en operaciones prácticas. Incluye ejercicios descargables y te ayudaremos a resolver dudas durante el curso.

Público Objetivo

Principiantes en Excel que deseen:

  • Mejorar sus habilidades laborales
  • Optimizar procesos de trabajo
  • Prepararse para exámenes o certificaciones
  • Gestionar datos personales o empresariales

Metodología

  • Enfoque 100% práctico: Aprende haciendo
  • Ejercicios reales: Casos aplicables a situaciones cotidianas
  • Soporte continuo: Resolución de dudas durante el curso
  • Material descargable: Plantillas y ejercicios para practicar

Lo que lograrás

  • Dominar las funciones básicas de Excel, mediante ejercicios prácticos
  • Crear tablas y gráficos profesionales
  • Aplicar fórmulas para automatizar tareas
  • Analizar y visualizar datos eficientemente
  • Certificado de finalización avalado

Temario Detallado

8 sesiones estructuradas para llevar tus habilidades de Excel al siguiente nivel

Módulo 1 – Libros y Hojas
Fundamentos
  • Insertar, nombrar y proteger hojas de trabajo
  • Manipulación de columnas, filas y celdas
  • Formatos básicos de celdas
  • Guardar y organizar libros de trabajo
Práctica: Múltiples ejercicios + Creación de un libro de gestión personal
Módulo 2 – Manipulando Datos
Operaciones básicas
  • Inserción y formato de datos
  • Estilos y auto-relleno inteligente
  • Buscar y reemplazar datos
  • Ocultar y mostrar elementos
Práctica: Múltiples ejercicios + Organización de una base de datos simple
Módulo 3 – Operaciones con Fórmulas Básicas
Cálculos esenciales
  • Operaciones aritméticas básicas
  • Funciones SUM, MAX, MIN
  • Cálculo de promedios
  • Funciones de redondeo
Práctica: Múltiples ejercicios + Cálculo de gastos mensuales
Módulo 4 – Visualización y Manejo
Análisis de datos
  • Comparar libros y ventanas
  • Formato condicional
  • Validación de datos
  • Referencias absolutas y relativas
Práctica: Múltiples ejercicios + Tablero de control básico
Módulo 5 – Tablas y Gráficos
Visualización profesional
  • Crear y formatear tablas
  • Filtros y ordenamiento de datos
  • Creación de gráficos profesionales
  • Propiedades y personalización de gráficos
Práctica: Múltiples ejercicios + Informe visual con gráficos dinámicos
Módulo 6 – Fórmulas Avanzadas
Automatización
  • Funciones de texto: MAYÚSC, CONCATENAR
  • CONTAR.SI y SUMAR.SI
  • Funciones SI anidadas
  • BUSCARV para cruce de datos
Práctica: Múltiples ejercicios + Sistema de búsqueda y cruce de información
Módulo 7 – Visualización Avanzada
Productividad
  • Exportar a PDF y otros formatos
  • Configuración de autoguardado
  • Protección de hojas y libros
  • Configuración de impresión
Práctica: Múltiples ejercicios + Preparación de reportes para impresión
Módulo 8 – Sesiones Extras y Proyecto Final
Aplicación real
  • Práctica en vivo con ejercicios financieros reales
  • Trucos y atajos personalizados
  • Resolución de problemas comunes
  • Proyecto final integrador
Proyecto Final: Desarrollo de un sistema completo de gestión

Nota: Cada sesión incluye tiempo dedicado para ejercicios prácticos y sesión de preguntas y respuestas (Q&A) para resolver dudas específicas.
